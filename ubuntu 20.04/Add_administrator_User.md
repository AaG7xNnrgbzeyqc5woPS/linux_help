# 1. ubuntu 20.04图形界面增加用户
- 1.1 左下角“显示应用程序”，找到“设置”
- 1.2 左面的 项目选择 框滑到最下面，倒数第四个，“用户”，选择
- 1.3 右上角“解锁”，输入密码，设置窗口的顶部，出现 一个绿色按钮“添加用户”
